## Decision
Product: chunkText module lives at apps/web/src/lib/chunk-text.ts (colocated with voice-utils, language-utils, similar-voices, etc.). Tests at apps/web/src/lib/chunk-text.test.ts. Function signature: `chunkText(text: string, maxLength: number): string[]`. Sentence boundaries defined as `.`, `!`, `?` followed by whitespace. Hard-split when a single sentence exceeds maxLength — break into maxLength-sized pieces. No content is ever dropped. Empty/whitespace-only input returns []. maxLength <= 0 throws.

## Decision
Placement: `apps/web/src/lib/chunk-text.ts` + `apps/web/src/lib/chunk-text.test.ts` — co-located with existing utilities like `language-utils.ts` / `provider-colors.ts` / `voice-utils.ts`. This follows the repo's pattern of flat utility modules in `src/lib/` alongside their `.test.ts` files. No new subdirectory needed (this is a single standalone function, not a subsystem).

## Decision
Naming: `chunkText` as specified. The module file: `chunk-text.ts` (kebab-case matches `language-utils.ts`, `provider-colors.ts`, `voice-utils.ts` convention). Test file: `chunk-text.test.ts` (matches vitest `include: ['src/**/*.test.ts']` glob).

## Decision
Test framework: vitest with `describe`/`it`/`expect` imported from 'vitest' — exactly matching all existing test files (`language-utils.test.ts`, `provider-colors.test.ts`, `ssml/tags.test.ts`, etc.). No additional test dependencies needed.
