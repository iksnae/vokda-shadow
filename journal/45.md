## Decision
Product: chunkText module lives at apps/web/src/lib/chunk-text.ts (colocated with voice-utils, language-utils, similar-voices, etc.). Tests at apps/web/src/lib/chunk-text.test.ts. Function signature: `chunkText(text: string, maxLength: number): string[]`. Sentence boundaries defined as `.`, `!`, `?` followed by whitespace. Hard-split when a single sentence exceeds maxLength — break into maxLength-sized pieces. No content is ever dropped. Empty/whitespace-only input returns []. maxLength <= 0 throws.
